# Team Sharing Setup

Use Obsidian Sync for the shared `Orbit Design Brain` vault.

## Setup Steps

1. Create or open this folder as an Obsidian vault named `Orbit Design Brain`.
2. Enable Obsidian Sync for the vault.
3. Invite design and product team members who need access.
4. Confirm each collaborator has active Sync access.
5. Ask contributors to edit canonical notes only in this vault.
6. Use `_review/Change Request Template.md` for proposed changes to stable files.

## Notes

Obsidian stores notes as plain Markdown files, so the vault remains portable. Obsidian's
team sync guidance says shared vault collaborators need active Sync access.

Reference: https://obsidian.md/help/teams/sync
