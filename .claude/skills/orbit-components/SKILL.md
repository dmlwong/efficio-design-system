# Orbit components

54 React components across 8 categories. Import everything
from the package boundary:

```tsx
import { Button, Card, Input } from '@efficio/orbit';
import type { ButtonProps } from '@efficio/orbit';
```

## Rules

- Import Orbit UI from `@efficio/orbit`, never from a local copy.
- **Compound components compose primitives — they never redefine token values.**
  If you need a new value, add a token (see orbit-tokens), don't hard-code it.
- Props are fully typed; the per-category reference files below list every prop,
  its type, whether it's required, and its default. Token values live in the
  **orbit-tokens** skill; accessibility contracts in **orbit-a11y**.
