# Orbit component descriptions

- **Alert** — Renders a multiline status alert with icon, title, optional description, and dismiss action.
- **Avatar** — Displays user initials in a styled circular avatar, with optional status and custom color.
- **AvatarStack** — Displays a compact row of avatars with optional overflow count.
- **Badge** — Displays a compact count or status badge in a semantic color.
- **Banner** — Back-compatible alias for Alert; use InlineBanner for Figma feedback strips.
- **Breadcrumb** — Renders a breadcrumb trail of links and the current page.
- **Button** — Renders an Orbit button with variant, size, state, and optional icons.
- **Carat** — Displays a small decorative caret indicator when visible.
- **Card** — Wraps content in a non-interactive Orbit card surface with configurable border and background treatment.
- **Checkbox** — Renders a labelled native checkbox with checked, disabled, and accessible-name support.
- **Chip** — Displays a static, toggleable, or removable compact chip with semantic styling.
- **CountryFlag** — Renders a country flag glyph for supported country codes.
- **CurrencyInput** — Renders a labelled currency input with prefix, validation states, and accessible naming.
- **DateInput** — Renders a date input with an optional picker button and accessible naming.
- **DocumentGlyph** — Displays a document-type glyph for files such as PDF, DOC, XLS, ZIP, and images.
- **Dropdown** — Renders a labelled select control with helper, error, and accessible naming support.
- **Dropzone** — Renders an accessible file dropzone with drag, choose-file, validation, and disabled states.
- **FaIcon** — Font Awesome 6 Pro icon component.
- **FileItem** — Displays a file row with name, meta content, optional icon, and optional action.
- **Filter** — Displays a removable filter pill with text and close action.
- **Headings** — Renders semantic heading text with Orbit heading sizes.
- **IconButton** — Renders an icon-only Orbit button with accessible label, variant, size, and state.
- **InlineBanner** — Renders a compact single-line feedback banner with semantic tone and optional dismiss action.
- **Input** — Renders a labelled text input with optional icon, helper text, error text, and accessible naming.
- **LegendLabel** — Displays a color marker with label text for legends and chart keys.
- **LinkText** — Renders an inline link styled with Orbit text and disabled states.
- **MultiSelectDropdown** — Renders a labelled multi-select control with helper, error, and accessible naming support.
- **MultiStateButton** — Renders one option inside a segmented multi-state button group.
- **MultiStateGroup** — Groups multi-state buttons as a labelled segmented control.
- **Overlay** — Renders an accessible modal overlay with focus management and configurable size.
- **PageHeader** — Renders an application page header with icon, title, subtitle, tabs, pills, and actions.
- **PriceIndicator** — Displays a directional price movement indicator with semantic icon and color.
- **QuickFilterGroup** — Groups quick filter items in a labelled navigation region.
- **QuickFilterItem** — Renders a compact quick filter button with active state and count.
- **RadialIndicator** — Displays radial progress with label, value, status, and clamped progress.
- **Radio** — Renders a labelled native radio input with disabled and accessible-name support.
- **RadioGroup** — Groups radio options in a labelled radiogroup container.
- **Required** — Displays the required-field marker used beside form labels.
- **RiskIndicator** — Displays a risk level with directional icon and semantic color.
- **Searchbox** — Renders a search input with submit button, disabled state, and accessible naming.
- **Separator** — Renders a horizontal or vertical visual separator.
- **SideNav** — Renders the Orbit application side navigation with brand, nav items, sections, work items, and profile area.
- **Spinner** — Displays an animated loading spinner with optional accessible label.
- **StatusIndicator** — Displays a small status dot with optional label text.
- **StepCircle** — Displays a workflow step marker in checked, numbered, active, or disabled states.
- **TabButton** — Renders a tab-style button with selected, disabled, and focus states.
- **Table** — Renders a data table with sortable headers, selectable rows, empty state, and pagination.
- **Text** — Renders text using Orbit body, emphasis, link, and semantic color styles.
- **TextArea** — Renders a labelled multiline text area with helper, error, and accessible naming support.
- **Textbox** — Renders a labelled text field with optional lock icon, helper text, and validation states.
- **Toast** — Renders a transient notification with semantic tone, optional dismiss button, and actions.
- **Toggle** — Renders a labelled switch control with disabled and accessible-name support.
- **ToolNextStepsCard** — Renders a card of recommended next-step actions with optional expandable table content.
- **Tooltip** — Wraps a trigger with hover/focus tooltip content and configurable placement.
